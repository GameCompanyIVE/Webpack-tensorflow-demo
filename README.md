
## How to use

First, install [Webpack](https://www.npmjs.com/package/webpack) and [webpack-dev-server](https://www.npmjs.com/package/webpack-dev-server) globally.

```bash
$ npm i -g webpack webpack-dev-server
```

Install the dependencies.

```bash
$ npm init
$ npm install
```

Run the project.

```bash
$ npm run dev
```
